# Inbox-killer


## Install :

1. apt update -y

2. apt upgrade -y

3. pkg install git -y

4. pkg install python -y

5. pip install requests

6. git clone https://github.com/Cyber-Foxx/Inbox-killer.git


### Run :

1. cd Inbox-killer

2. python 'Inbox killer.py'

#### Single Code For Termux :

cd $HOME && cd Inbox-killer && python Inbox killer.py
